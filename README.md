StructLens is a browser extension that reveals structured data hidden inside modern websites.

It helps developers, analysts and data engineers understand where data comes from in any webpage.

## Features

### Free (local)

* Detect HTML tables
* Extract embedded JSON (ld+json, NEXT_DATA, INITIAL_STATE)
* Inspect network requests (XHR / fetch)
* Parse schema.org structured data
* Export to CSV or JSON

### Pro (cloud powered)

* Scraper generator
* Pagination detection
* Data monitoring
* Integrations (Sheets, Notion, Webhooks)

## Tech Stack

* TypeScript
* React
* Manifest V3

## Philosophy

Free inspection. Paid automation.
